배송 준비 중 상태 이전까지는 마이페이지에서 직접 취소 가능합니다.

취소 방법:

  1. 마이페이지 > 주문 내역
  2. 취소할 주문 선택
  3. '주문 취소' 버튼 클릭

배송 준비 중 이후 단계에서는 고객센터(1544-0000)로 문의해주세요.

주문 후 영업일 기준 2-3일 내 배송됩니다.

배송비:

  • 30,000원 이상 구매 시: 무료배송
  • 30,000원 미만 구매 시: 3,000원

제주도 및 도서산간 지역은 추가 배송비가 발생할 수 있습니다.

상품 수령 후 7일 이내 교환/반품 신청 가능합니다.

신청 방법:

  1. 마이페이지 > 주문 내역
  2. 해당 주문에서 '교환/반품 신청'
  3. 사유 선택 및 상세 내용 입력

주의사항:

  • 상품 훼손 시 교환/반품 불가
  • 단순 변심의 경우 왕복 배송비 고객 부담

다양한 결제 수단을 지원합니다:

  • 신용카드: 국내 모든 카드사
  • 간편결제: 카카오페이, 네이버페이, 토스
  • 계좌이체: 실시간 계좌이체
  • 무통장입금: 주문 후 24시간 내 입금

할부는 5만원 이상 구매 시 가능합니다.

로그인 페이지에서 비밀번호를 재설정할 수 있습니다.

재설정 방법:

  1. 로그인 페이지 > '비밀번호 찾기'
  2. 가입 시 등록한 이메일 입력
  3. 이메일로 발송된 링크 클릭
  4. 새 비밀번호 설정

이메일을 확인할 수 없는 경우 고객센터로 문의해주세요.

배송 준비 중 이전까지만 변경 가능합니다.

변경 방법:

  1. 마이페이지 > 주문 내역
  2. 해당 주문 선택
  3. '배송지 변경' 버튼 클릭

배송 시작 후에는 택배사에 직접 연락하여 변경해주세요.

구매 금액의 1%가 자동 적립됩니다.

적립 시기:

  • 배송 완료 후 7일 경과 시 적립
  • 리뷰 작성 시 추가 500P 적립

사용 방법:

  • 1,000P 이상부터 사용 가능
  • 결제 시 자동 차감

반품 상품 확인 후 영업일 기준 3-5일 내 처리됩니다.

환불 방법:

  • 신용카드: 결제 취소 (카드사별 2-7일 소요)
  • 간편결제: 해당 서비스로 환불 (1-3일 소요)
  • 무통장입금: 등록 계좌로 환불 (2-3일 소요)
